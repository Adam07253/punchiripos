# 🚀 Punchiri POS - Quick Start

## ⚡ For Development

```bash
npm start
```

That's it! App launches with server auto-starting.

---

## 📦 For Building

```bash
npm run build
```

Output: `dist/Punchiri POS Setup.exe` and `dist/latest.yml`

---

## 🌐 For GitHub Release

1. Go to: https://github.com/Adam07253/punchiri-pos/releases
2. Click "Create a new release"
3. Tag: `v1.0.0`
4. Upload both files from `dist/` folder
5. Click "Publish release"

---

## 🔄 For Updates

1. Edit `package.json`: Change version to `1.0.1`
2. Run: `npm run build`
3. Create new GitHub release: `v1.0.1`
4. Upload new files
5. Users get auto-update!
